# 国内网址部署准备清单

本项目当前是纯静态官网，可以部署到国内云服务器、对象存储静态网站、CDN 或传统虚拟主机。GitHub 只建议作为代码仓库，不建议用 GitHub Pages 作为国内正式访问入口。

## 推荐部署路线

1. 购买并实名认证域名
   - 建议使用国内有资质的注册服务商。
   - 域名主体信息要和后续备案主体保持一致。

2. 选择国内接入商
   - 可选阿里云、腾讯云、华为云等中国内地节点。
   - 静态官网可优先选轻量服务器、ECS/CVM + Nginx，或对象存储 + CDN。

3. 办理 ICP 备案
   - 域名解析到中国内地服务器并对外提供 Web 服务时，需要先完成 ICP 备案。
   - 备案通常通过服务器所在接入商提交。
   - 备案通过前不要开放未备案域名访问。

4. 上线 HTTPS
   - 使用云厂商免费证书或商业证书。
   - 配置 HTTP 自动跳转 HTTPS。

5. 办理公安联网备案
   - 网站上线后，按要求在全国互联网安全管理服务平台提交公安联网备案。
   - 备案号需要展示在页脚。

6. 配置 CDN 与监控
   - 域名已有 ICP 备案后再接入中国内地 CDN。
   - 接入访问统计、错误日志、表单通知和基础可用性监控。

## 上线前必须替换

- 工作室主体信息或个人主体名称
- 工作室主体信息
- ICP 备案号：`湘ICP备2026020536号`
- 公安联网备案号
- 隐私政策、用户信息收集说明
- 真实业务资质、已授权案例和服务边界

## 静态站点部署方式

### Nginx

把仓库中的以下文件上传到服务器站点目录：

- `index.html`
- `privacy.html`
- `styles.css`
- `main.js`
- `robots.txt`
- `sitemap.xml`
- `README.md` 可不上传
- `assets/`

参考配置见 `deploy/nginx-cn-static.conf.example`。

### 对象存储 + CDN

适合纯静态官网：

- 阿里云 OSS + CDN
- 腾讯云 COS + CDN
- 华为云 OBS + CDN

需要注意：如果 CDN 使用中国内地节点，域名通常仍需完成 ICP 备案。
